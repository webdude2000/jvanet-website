# Project: JVANET Corporate Website

## What this does
Corporate website for JVANET PTY LTD — Australian technology company building AI-powered products and offering enterprise IT consulting. Single-page Next.js site deployed on Vercel.

## Architecture
- Next.js 14+ App Router, TypeScript, Tailwind CSS
- Single page with section-based smooth scroll navigation
- Static site — no database, auth, or API routes
- Deployed to Vercel, domain: jvanet.com

## Current focus
Initial build and deployment — get the site live with all core sections.
